Methodology

How the index works

Each site is crawled and scored by agent-readiness-auditor, an open source instrument published on npm, across nine checks. Hidden prompt injection text, llms.txt, the robots.txt stance on AI crawlers, structured data, accountability links, a sitemap, content structured for AI answers, accidental index blocking, and whether an agent endpoint such as an MCP server is advertised. Prompt injection hygiene carries the most weight because it is a safety property.

The panel holds 200 prominent sites across twelve sectors, chosen for reach and influence. The first 84 carry over from The State of the Agentic Web study so results stay comparable across editions. Anyone can reproduce any score by running npx agent-readiness-auditor example.com. The panel list and raw results for this edition are published in the open.
