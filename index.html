Quarterly benchmark · Q3 2026

The Agentic Web Index

200 prominent websites, ranked by how ready they are for the AI assistants and autonomous agents that increasingly read the web on people's behalf.

Every site is scored 0 to 100 by the open source agent readiness auditor across nine automated checks. The full methodology and data are public and reproducible. Published July 2026, updated quarterly.

Headline findings

The web is still not ready for AI agents

76/100median score
38%publish llms.txt
25%give AI crawlers explicit rules
24%block automated visitors entirely

Of the 200 sites on the panel, 152 could be audited and 48 refused automated visitors outright, making themselves invisible to AI assistants. The strongest sites this edition are coursera.org and samsung.com at 100/100. Only 13% of auditable sites publish content structured in the formats AI answers quote most readily.

Methodology

How the index works

Each site is crawled and scored by agent-readiness-auditor, an open source instrument published on npm, across nine checks. Hidden prompt injection text, llms.txt, the robots.txt stance on AI crawlers, structured data, accountability links, a sitemap, content structured for AI answers, accidental index blocking, and whether an agent endpoint such as an MCP server is advertised. Prompt injection hygiene carries the most weight because it is a safety property.

The panel holds 200 prominent sites across twelve sectors, chosen for reach and influence. The first 84 carry over from The State of the Agentic Web study so results stay comparable across editions. Anyone can reproduce any score by running npx agent-readiness-auditor example.com. The panel list and raw results for this edition are published in the open.
